Usage: rsg [input] [options]

Options:
  -o, --output     Output directory                      [default: 'styleguide']
  -t, --title      Used as a page title                  [default: 'Style Guide']
  -p, --pushState  Enable HTML5 pushState                [default: false]
  -v, --verbose    Verbose output                        [default: false]
  -b, --basePath   Set the base path                     [default: null]
  -f, --files      Inject references to files into HTML  [default: null]
  -c, --config     wip                                   [default: './react-styleguide.json']

Examples:
  rsg 'example/**/*.js' -t 'Cool Components'
